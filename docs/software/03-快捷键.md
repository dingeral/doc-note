## Win11 功能快捷键

|      |    快捷键  |        执行此操作      |
|:-------------|:------------------|:-------------|
| 1    |   Win + A        | 打开快速设置面板 |
| 2    |   Win + I         |   打开设置      |
| 3    |   Ctrl + X        |   剪切选定项     |
| 4    |   Ctrl + C        |   复制选定项      |
| 5    |   Ctrl + V        |   粘贴选定项      |
| 6    |   Ctrl + Z        |   撤消操作      |
